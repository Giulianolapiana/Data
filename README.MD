# DataAnalist

Repositorio de trabajos prácticos de **Análisis de Datos**.

Este espacio reúne los notebooks y datasets utilizados durante la cursada para practicar:

- limpieza y preparación de datos,
- análisis exploratorio (EDA),
- visualización,
- interpretación de resultados,
- elaboración de reportes.

## Trabajos prácticos

### TP2

Notebook introductorio del flujo de análisis de datos: carga, inspección inicial y primeras transformaciones.

### TP3

Profundización en limpieza y normalización de datos, tratamiento de nulos y consistencia de tipos.

### TP4

Análisis exploratorio con estadísticas descriptivas y visualizaciones para detectar patrones y outliers.

### TP5

Análisis orientado a métricas y comparación de variables para apoyar la toma de decisiones.

### TP6

Integración de técnicas vistas en la materia para resolver un caso más completo de análisis.

### TPI

Trabajo Práctico Integrador donde se consolidan los contenidos de la cursada.

## Datasets incluidos

- `notas_semestre.csv`: datos académicos para ejercicios de análisis de rendimiento.
- `productos.csv`: datos de productos para prácticas de limpieza, segmentación y análisis descriptivo.
- `reporte_elearning.csv`: información de e-learning para análisis de actividad y resultados.

## Requisitos sugeridos

- Python 3.10+
- Jupyter Notebook o VS Code con extensión Jupyter

## Cómo usar este repositorio

1. Abrir la carpeta `DataAnalist` en VS Code.
2. Ir a `TrabajosPracticos`.
3. Abrir el notebook que quieras ejecutar (`TP2.ipynb`, `TP3.ipynb`, etc.).
4. Seleccionar kernel de Python y ejecutar las celdas en orden.

## Objetivo académico

Documentar la evolución práctica en análisis de datos, aplicando conceptos de la materia sobre casos reales y semi-reales con evidencia reproducible en notebooks.
